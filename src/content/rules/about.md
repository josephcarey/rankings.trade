# About rankings.trade

**rankings.trade** is a competitive leaderboard for [SpaceTraders](https://spacetraders.io)
agents. It tracks public agent performance, groups players into leagues, and recognises
standout agents over time.

These help pages explain the core concepts so newcomers can get oriented. Use the index at
[/rules](/rules) to browse topics.

> Some features described here are still being rolled out. Where a page describes how a rule
> works (rather than a specific button to click), it reflects the intended behaviour.
