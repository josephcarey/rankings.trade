# Security Policy

rankings.trade is a live, publicly deployed application. We take security
reports seriously and appreciate responsible disclosure.

## Supported versions

Only the latest deployed `main` branch is supported. There are no released
versions or backports — fixes ship by deploying `main`.

| Version | Supported |
| ------- | --------- |
| latest deployed `main` | ✅ |
| anything else | ❌ |

## Reporting a vulnerability

**Please do not open a public issue for security vulnerabilities.**

Report privately through GitHub's
[**Report a vulnerability**](https://github.com/josephcarey/rankings.trade/security/advisories/new)
flow (private security advisories). This keeps the details confidential until a
fix is available.

When reporting, please include:

- A description of the vulnerability and its potential impact.
- Steps to reproduce (proof-of-concept where possible).
- Any affected URLs or components.

## What to expect

This is a best-effort hobby project with **no formal SLA**. We'll acknowledge
and investigate reports as time permits and credit reporters who wish to be
named once a fix is deployed.
